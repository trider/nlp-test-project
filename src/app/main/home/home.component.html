To start a chat

  1. Enter your name and phone number
  2. In the Customer Agent Prompt, edit and expand the placeholder text
  3. Click on the tags to add them to the prompt
  4. Create your own tags, by surrounding a single word or phrase with two sets of square brackets,[[My Tag]], and press return
  5. Click Submit. An alert will show you all the data you submitted
